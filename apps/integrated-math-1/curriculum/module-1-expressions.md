# Module 1: Expressions

## Overview
This module establishes the foundational language of algebra through expressions. Students transition from evaluating numerical expressions with order of operations to writing and interpreting algebraic expressions with variables. The module emphasizes mathematical precision—translating between words and symbols, naming properties correctly, and recognizing when expressions model real-world situations. These skills prepare students for the equation work that begins in Module 2.

## Skills Developed
- Translate verbal descriptions into numerical and algebraic expressions using operation symbols and grouping
- Evaluate expressions using the order of operations, including powers, nested grouping, and fractions
- Identify and apply properties of equality (reflexive, symmetric, transitive) and properties of real numbers (commutative, associative, identity, inverse)
- Use the Distributive Property to rewrite and simplify expressions; combine like terms
- Write and evaluate absolute value expressions for real-world accuracy and tolerance
- Apply ratios and percentages as metrics; determine reasonable levels of accuracy in multi-step calculations
