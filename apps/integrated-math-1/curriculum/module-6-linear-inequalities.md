# Module 6: Linear Inequalities

## Overview
This module extends equation-solving skills to inequalities, beginning with one-step inequalities and progressing to compound and absolute value inequalities. Students graph solution sets on number lines, translate real-world constraints into algebraic inequalities, and graph linear inequalities in two variables on the coordinate plane. The module emphasizes reasoning about solution sets and understanding when inequality solutions differ from equation solutions.

## Skills Developed
- Solve one-step inequalities involving addition, subtraction, multiplication, and division; reverse the inequality symbol when multiplying or dividing by a negative number
- Graph solution sets on a number line using open and closed circles appropriately
- Solve multi-step inequalities using the Distributive Property and combining like terms; recognize special solution sets (no solution, all real numbers)
- Solve compound inequalities involving "and" (intersection) and "or" (union); rewrite three-part inequalities and graph the solution
- Solve absolute value inequalities by rewriting them as compound inequalities; recognize special cases with no solution or all real numbers
- Model real-world situations with absolute value inequalities for tolerance and margin-of-error contexts
- Graph linear inequalities in two variables; identify the boundary line, test points, and half-plane; write inequalities from graphs
