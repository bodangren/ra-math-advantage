# Module 4: Linear and Nonlinear Functions

## Overview
This module deepens the study of linear functions and introduces nonlinear function families. Students graph linear equations using tables and intercepts, compute slope and rate of change, and write equations in slope-intercept, point-slope, and standard forms. The module also covers transformations of linear functions, arithmetic sequences as discrete linear functions, piecewise and step functions, and absolute value functions with their characteristic V-shaped graphs.

## Skills Developed
- Graph linear equations by making a table of values and by plotting x- and y-intercepts; interpret intercepts in context
- Calculate rate of change and slope from tables, graphs, and two points; determine linearity from constant rate of change
- Write and graph equations in slope-intercept form; rewrite standard form into slope-intercept form
- Describe translations, dilations, and reflections of linear and absolute value functions relative to their parent functions
- Identify arithmetic sequences, find the common difference, write explicit formulas, and model real-world situations
- Graph piecewise-defined and step functions; identify domain and range
- Graph absolute value functions, state domain and range, and write equations from graphs
