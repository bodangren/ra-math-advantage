# Module 2: Equations in One Variable

## Overview
This module focuses on solving equations in one variable, building from one-step equations to multi-step equations with variables on both sides. Students learn to translate between verbal sentences and algebraic equations, apply Properties of Equality to isolate variables, and interpret solutions in context. The module concludes with proportions and formulas—tools that connect algebraic manipulation to real-world measurement and modeling.

## Skills Developed
- Translate verbal sentences and real-world situations into algebraic equations and formulas
- Solve one-step equations using inverse operations and Properties of Equality
- Solve multi-step equations by undoing operations in reverse order; check solutions by substitution
- Solve equations with variables on both sides; classify equations as having one solution, no solution, or infinitely many solutions (identities)
- Solve absolute value equations and interpret solutions in tolerance and margin-of-error contexts
- Solve proportions using the Cross Products Property; apply to rates, scales, and percent problems
- Solve literal equations and formulas for a specified variable; use dimensional analysis for unit conversion
