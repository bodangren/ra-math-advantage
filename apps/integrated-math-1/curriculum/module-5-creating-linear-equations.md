# Module 5: Creating Linear Equations

## Overview
This module focuses on constructing linear equations to model data and real-world situations. Students write equations given a point and slope, two points, or a verbal description, and they learn to convert between slope-intercept, point-slope, and standard forms. The module covers scatter plots, lines of fit, and linear regression for modeling bivariate data, along with the critical distinction between correlation and causation. The module concludes with inverses of linear functions.

## Skills Developed
- Write linear equations in slope-intercept form given a point and slope or given two points
- Write and convert equations between point-slope, slope-intercept, and standard forms; identify parallel and perpendicular lines from slopes
- Analyze scatter plots to identify positive, negative, or no correlation; interpret in context
- Write equations of lines of fit and use them to make predictions; distinguish interpolation from extrapolation
- Use technology to calculate linear regression equations and correlation coefficients; interpret r in context
- Analyze residual plots to assess the appropriateness of a linear model
- Distinguish correlation from causation; identify lurking variables
- Find inverses of linear functions algebraically (switch-and-solve method) and graphically (reflection across y = x)
