This document is the audited class-period breakdown for Module 4.

It should be treated as the canonical planning source for Module 4 class periods unless a later audit supersedes it.

It follows the planning model in `curriculum/course-spec.md`:

- the **class period** is the atomic planning unit
- `instruction` days must map to `measure/course-objectives.md`
- `instruction` days must name the specific worked examples they use
- `mastery`, `jigsaw`, `review`, and `test` days are marked primarily by day type

## Notes

- This is a **planning model**, not a literal calendar.
- Most instruction days use **2-3 worked examples**.
- Module 4 is the longest IM1 module with 7 lessons; it establishes core linear function graphing and equation-writing skills.
- Lessons 4-4 (transformations), 4-6 (piecewise/step), and 4-7 (absolute value functions) extend beyond basic linear functions and benefit from the extended instructional time.
- Arithmetic sequences (Lesson 4-5) connect directly to linear functions through constant rate of change.

## Why This Split Works

- `4-1` splits graphing method (4a) from contextual interpretation (4b), keeping these distinct skills on separate days.
- `4-2` is the densest lesson; it gets three days to separate rate-of-change computation (4c), linearity checking and slope (4d), and real-world slope interpretation (4e).
- `4-3` separates writing and rewriting equations (4f) from modeling and graphing (4g), with the jigsaw placed between these two to consolidate slope-intercept form fluency.
- `4-4` splits into translations (4h) and dilations/reflections (4i), reflecting the two major transformation categories.
- `4-5` separates arithmetic sequence identification (4j) from explicit formula and modeling (4k), which builds on the identification skill.
- `4-6` splits piecewise functions (4l) from writing and modeling with step functions (4m), keeping graph analysis separate from application.
- `4-7` splits description/graphing (4n) from writing equations from graphs and modeling (4o), using the full arc of absolute value function work.
- The regular mastery and jigsaw days maintain engagement and retention across this long module.
