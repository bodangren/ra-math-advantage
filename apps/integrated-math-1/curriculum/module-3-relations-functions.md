# Module 3: Relations and Functions

## Overview
This module introduces students to the concept of a function as a precise mathematical relationship between inputs and outputs. Students represent relations in multiple forms—ordered pairs, tables, graphs, and mappings—and develop vocabulary for describing function behavior. The module emphasizes interpreting graphs in context: intercepts, intervals of increase and decrease, extrema, symmetry, and end behavior all emerge from real-world data.

## Skills Developed
- Represent relations as ordered pairs, tables, graphs, and mappings; identify domain and range
- Distinguish independent and dependent variables; interpret axes, scale, and origin in context
- Determine whether a relation is a function using the vertical line test, ordered pair analysis, and equation solving
- Evaluate functions using function notation; interpret function values in real-world contexts
- Classify functions as discrete, continuous, or neither; determine linearity from equations, graphs, and tables
- Find and interpret x- and y-intercepts; locate zeros of linear functions
- Identify intervals of increase and decrease, relative maxima and minima, line symmetry, and end behavior from graphs
- Sketch graphs of linear and nonlinear functions from verbal descriptions; compare two functions by their key features
