# Module 7: Systems of Linear Equations and Inequalities

## Overview
This module introduces systems of equations and inequalities as tools for modeling situations where two or more conditions must hold simultaneously. Students graph systems to identify solutions, then develop algebraic techniques—substitution and elimination (by addition, subtraction, and multiplication)—to solve systems exactly. The module closes with systems of inequalities, where the solution is the overlapping region satisfying all constraints.

## Skills Developed
- Graph systems of linear equations and determine the number of solutions; classify systems as consistent/inconsistent and independent/dependent
- Solve systems using the substitution method when one variable is isolated or when both equations are solved for the same variable
- Solve systems using elimination by addition when coefficients are opposites and by subtraction when coefficients are identical
- Solve systems using elimination with multiplication when neither coefficients are already opposites or identical; choose strategically which variable to eliminate
- Write and solve systems of equations to model real-world situations involving mixtures, rates, geometry, and finance
- Solve linear equations by rewriting them as a system of two equations and graphing
- Graph systems of linear inequalities; identify the feasible region as the overlap of all half-planes
- Write systems of inequalities from graphs and determine whether ordered pairs satisfy the system
