# Module 8: Exponential Functions

## Overview
Exponential functions model situations where quantities grow or decay by a constant percent rate. This module builds from recognizing exponential patterns in data, through analyzing and transforming exponential functions, to connecting exponential functions with geometric sequences. Students learn to write exponential equations from two points, compare rates with different compounding periods, and work with recursive and explicit forms of sequences.

## Skills Developed
- Identify exponential behavior using the common ratio test
- Graph exponential functions and identify key features (domain, range, asymptote, initial value, growth/decay factor)
- Describe and apply transformations of exponential functions
- Write exponential functions from two points using the two-point method
- Apply the compound-interest formula and compare rates with different compounding periods
- Find effective annual, quarterly, and monthly rates using properties of exponents
- Identify geometric sequences and find terms using explicit and recursive formulas
- Convert between recursive and explicit forms for arithmetic and geometric sequences
