## CED Learning Objectives

- LO 3.11.A: Define and analyze secant, cosecant, and cotangent as reciprocal functions. Identify vertical asymptotes and ranges. Solve equations involving reciprocal trig functions.

## CED Essential Knowledge

- EK 3.11.A.1: Secant is the reciprocal of cosine: sec x = 1/cos x, where cos x ≠ 0. Cosecant is the reciprocal of sine: csc x = 1/sin x, where sin x ≠ 0. Cotangent is the reciprocal of tangent: cot x = 1/tan x = cos x / sin x, where sin x ≠ 0.
- EK 3.11.A.2: The graphs of reciprocal functions have vertical asymptotes wherever the denominator is zero. sec x has asymptotes at x = π/2 + kπ. csc x has asymptotes at x = kπ. cot x has asymptotes at x = kπ. The ranges of sec x and csc x are (−∞, −1] ∪ [1, ∞).
- EK 3.11.A.3: To solve equations involving reciprocal trig functions, isolate the trig function first, then use reciprocal properties to rewrite in terms of sine, cosine, or tangent.

## Passwater Scaffolding Notes

Passwater introduces reciprocal functions with clear fill-in-the-blank definitions: "Secant is the reciprocal of the cosine function." sec x = 1/cos x. "Cosecant is the reciprocal of the sine function." csc x = 1/sin x. cot x = 1/tan x = cos x / sin x.

Graph properties are tied directly to the reciprocal relationship: "The graphs of each of these reciprocal functions will include vertical asymptotes whenever the denominator is 0." This means: sec x has vertical asymptotes where cos x = 0 (x = π/2 + kπ), csc x where sin x = 0 (x = kπ), cot x where sin x = 0 (x = kπ). The ranges of sec x and csc x are both (−∞, −1] ∪ [1, ∞) because they are reciprocals of values in [−1, 1].

The solution strategy is explicit: "To do this, it is generally best to isolate the trig function first in the equation. Once the trig function is isolated, we can use the reciprocal properties to rewrite the equation in terms of sine, cosine, or tangent." This converts unfamiliar sec/csc/cot equations into familiar sin/cos/tan problems.

Scaffolding moves: identify asymptotes from equations (Examples 1–4) → find range (Example 3: h(θ) = 3 csc(θ/2), range (−∞, −3] ∪ [3, ∞)) → intersection problems with horizontal lines (Examples 5–7).

## Guided Practice

**Example 1:** f(x) = 3 sec(2x). Vertical asymptote when cos(2x) = 0 → 2x = π/2 + kπ → x = π/4 + kπ/2. At x = π/4 (one of the options).

**Example 2:** g(x) = 4 csc(2x − π). Asymptote when sin(2x − π) = 0 → 2x − π = kπ → x = π/2 + kπ/2. At x = π/2.

**Example 3:** h(θ) = 3 csc(θ/2). Range: since csc has range (−∞, −1] ∪ [1, ∞), multiplying by 3 gives (−∞, −3] ∪ [3, ∞).

**Example 5:** f(x) = 4 csc(3x) + 1, g(x) = 11. 4 csc(3x) + 1 = 11 → csc(3x) = 5/2 → sin(3x) = 2/5. Solve on [0, 2π).

## Additional Notes

The reciprocal relationship means sec x and csc x are undefined wherever cos x = 0 and sin x = 0 respectively — these create the vertical asymptotes. Between consecutive asymptotes, sec x and csc x each trace a U-shaped or inverted U-shaped curve (similar to a parabola) because they are the reciprocal of a function that passes through zero. The ranges (−∞, −1] ∪ [1, ∞) exclude (−1, 1) because |sin x| ≤ 1 and |cos x| ≤ 1, so their reciprocals have magnitude at least 1.

The intersection problems (e.g., 4 csc(3x) + 1 = 11) require converting to sin/cos, solving the resulting equation, and checking that the original reciprocal function is defined at the solution points. This connects Topics 3.10 and 3.11.

The quiz after Topic 3.11 tests asymptote identification, range determination, and solving reciprocal equations — both with and without calculator.
