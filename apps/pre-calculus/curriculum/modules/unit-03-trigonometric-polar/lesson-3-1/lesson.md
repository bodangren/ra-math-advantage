## Vocabulary

### Section: text

- **Periodic relationship**: A relationship between two variables that occurs when the output values demonstrate a repeating pattern over successive equal-length intervals.
- **Period**: The length of the input values over which the function completes one full cycle. If [f] is periodic with period [P], then [f(x) = f(x + P)] for all [x].
- **Cycle**: One complete repetition of the pattern within a periodic function.

A periodic function starts with a basic pattern and translates that pattern repeatedly over its domain. The defining property is [f(x) = f(x - P)] for all values of [x].

## Learn

### Section: text

A function is **periodic** if its output values repeat at regular intervals. The distance over which one full repetition occurs is the **period** [P]. For any periodic function [f] with period [P]:

\[f(x) = f(x + P) = f(x - P) = f(x + kP)\]

for any integer [k]. This means you can evaluate a periodic function at any input by shifting the input by multiples of the period until it falls within a known interval.

To find the period from a graph, measure the minimum horizontal distance between two points where the graph begins repeating — that is, the least positive value [P] such that the graph on [[a, a+P]] matches the graph on [[a+P, a+2P]].

Periodic functions model real-world cyclical phenomena: rotating objects, oscillating motion, seasonal temperature variation, tidal patterns, and sound waves.

## Worked Example

### Section: text

**Example 1:** The function [g] is periodic. A portion of its graph is shown on the interval [[0, 6]]. The graph rises from a minimum of [2] at [x = 0] to a maximum of [8] at [x = 3], then falls back to [2] at [x = 6]. What is the least possible value of the period?

**Solution:** The graph completes one full cycle from [x = 0] to [x = 6]. The least possible period is [P = 6].

**Example 2:** Using [g] from Example 1 with period 6, find [g(14)], [g(72)], and [g(-17)].

**Solution:**
- [g(14)]: Shift by multiples of 6. [14 - 12 = 2], so [g(14) = g(2)]. From the graph, [g(2) = 6] (midpoint of the rise).
- [g(72)]: [72 = 12 \times 6], so [g(72) = g(0) = 2].
- [g(-17)]: [-17 + 18 = 1], so [g(-17) = g(1)].

**Example 3:** The function [h] is periodic with period 5. Values are given: [h(1) = -2], [h(3) = 0], [h(4) = 3], [h(7) = 2]. Find [h(-2)] and [h(h(9))].

**Solution:**
- [h(-2)]: [-2 + 5 = 3], so [h(-2) = h(3) = 0].
- [h(h(9))]: First find [h(9)]. Since [9 - 5 = 4], [h(9) = h(4) = 3]. Then [h(3) = 0]. So [h(h(9)) = 0].

## Reflection

### Section: text

**Exit Ticket:** The function [f] is periodic with period 6 and [f(2) = 5]. Find [f(14)] and [f(-4)].

**Lesson Summary:** Periodic functions repeat their output pattern at regular intervals. The period [P] is the minimum distance for one full cycle, and the key property [f(x) = f(x + P)] lets you evaluate at any input by shifting to a known interval. Real-world rotating, oscillating, and seasonal phenomena are all modeled by periodic functions.
