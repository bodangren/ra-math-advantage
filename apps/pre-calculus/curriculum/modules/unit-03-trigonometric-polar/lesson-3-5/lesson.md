# Lesson 3-5 — Sinusoidal Functions

Unit: 3
Topic: 3.5
AP Exam: assessed-on-ap-exam

## Explore

### Section: text

Is the cosine function just another version of the sine function? Consider [\cos \theta = \sin(\theta + \pi/2)]. Graph both on the same axes. How are they related?

Every function that can be obtained by stretching, compressing, and shifting [\sin \theta] (or equivalently [\cos \theta]) is called **sinusoidal**. But not every repeating pattern is sinusoidal — a square wave repeats but is not sinusoidal. What makes sinusoidal functions special?

## Vocabulary

### Section: text

- **Sinusoidal function**: Any function that can be written as an additive and multiplicative transformation of [f(\theta) = \sin \theta]. Equivalently, any function of the form [a\sin(b\theta) + d] or [a\cos(b\theta) + d].
- **Increasing at an increasing rate**: The function is rising and the slope is getting steeper (concave up, above midline going up... or below midline going up — specifically concave up and increasing).
- **Increasing at a decreasing rate**: The function is rising but the slope is flattening (concave down and increasing).
- **Decreasing at an increasing rate**: The function is falling and the slope is becoming more negative (concave down and decreasing).
- **Decreasing at a decreasing rate**: The function is falling but the slope is flattening toward zero (concave up and decreasing).

## Learn

### Section: text

A **sinusoidal function** is any function that involves additive and multiplicative transformations of [\sin \theta]. The cosine function qualifies because:

\[\cos \theta = \sin\left(\theta + \frac{\pi}{2}\right)\]

Every sinusoidal function has four defining characteristics:
1. **Midline** — the vertical center of oscillation
2. **Amplitude** — the distance from midline to peak
3. **Period** — the horizontal length of one full cycle
4. **Frequency** — the number of cycles per unit of input ([1/\text{period}])

A sinusoidal function is always periodic, but the converse is not true. A square wave or sawtooth wave is periodic but not sinusoidal. Only smooth, wave-shaped oscillations that match the sine/cosine pattern are sinusoidal.

Describing interval behavior requires four phrases: "increasing at an increasing rate," "increasing at a decreasing rate," "decreasing at an increasing rate," and "decreasing at a decreasing rate." These combine direction (increasing/decreasing) with concavity (rate of change speeding up or slowing down).

## Worked Example

### Section: text

**Example 1:** A sinusoidal function [h(\theta)] has a maximum at [(\pi, 14)] and a minimum at [(3\pi/2, 6)]. Find the midline, amplitude, period, and frequency.

**Solution:**
- Midline: [d = (14 + 6)/2 = 10]
- Amplitude: [a = (14 - 6)/2 = 4]
- Period: [P = 2(3\pi/2 - \pi) = 2(\pi/2) = \pi]
- Frequency: [f = 1/\pi]

**Example 2:** On the interval [(0, \pi/2)], describe the behavior of [f(\theta) = \sin \theta].

**Solution:** [f] is increasing (from [0] toward [1]). The graph is above the midline and concave down as it approaches the peak. So [f] is **increasing at a decreasing rate**.

**Example 3:** Is every periodic function sinusoidal? Give a counterexample if not.

**Solution:** No. A square wave that alternates between [1] and [-1] at regular intervals is periodic but has sharp corners — it is not a smooth transformation of sine or cosine.

## Guided Practice

### Section: text

**Problem 1:** A sinusoidal function [g] has a maximum at [(\pi/3, 9)] and a minimum at [(2\pi/3, 1)]. Find the midline and amplitude.

**Hint:** Midline = [(max + min)/2], Amplitude = [(max - min)/2].

**Problem 2:** On the interval [(\pi/2, \pi)], is [\sin \theta] increasing or decreasing? Is it concave up or concave down? Combine these into one phrase.

**Hint:** Sine is falling from [1] to [0] (decreasing). It is above the midline and bending downward (concave down). So: decreasing at a decreasing rate.

**Problem 3:** Explain why [\cos \theta] is a sinusoidal function but [f(\theta) = \sin(\theta^2)] is not.

**Hint:** [\cos \theta = \sin(\theta + \pi/2)] — it is a horizontal shift of sine. [\sin(\theta^2)] has a changing period and is not a simple multiplicative/additive transformation.

## Reflection

### Section: text

**Exit Ticket:** A function has a maximum at [(\pi, 14)] and a minimum at [(3\pi/2, 6)]. Find the midline and amplitude. Is this function sinusoidal?

**Lesson Summary:** Sinusoidal functions are transformations of sine — they have smooth, wave-shaped graphs with a definite midline, amplitude, period, and frequency. Every sinusoidal function is periodic, but not every periodic function is sinusoidal. Describing behavior on sub-intervals requires combining direction (increasing/decreasing) with concavity (rate of change speeding up or slowing down).
