# Unit 4: Functions Involving Parameters, Vectors, and Matrices

## Overview

This unit covers parametric functions, implicit functions, conic sections, vectors, vector-valued functions, matrices, and linear transformations. Unit 4 is CED-defined but not assessed on the AP Exam.
