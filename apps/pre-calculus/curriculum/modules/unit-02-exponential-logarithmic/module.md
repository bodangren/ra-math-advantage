# Unit 2: Exponential and Logarithmic Functions

## Overview

This unit covers exponential and logarithmic functions, including sequences, growth/decay models, function composition, inverses, logarithmic properties, and semi-log plots.

## AP Exam Status

- Assessed on AP Exam
- Multiple-choice weighting: 27-40%
- Suggested pacing: 6-9 weeks
