# Unit 7: Probability

## Overview

Students develop understanding of counting principles, probability models, permutations, combinations, compound probability, and expected value for making fair decisions.

## Lessons

### 7-1 Basic Probability

Experimental vs theoretical probability, sample spaces, favorable outcomes, and complementary events.

### 7-2 Fundamental Counting Principle and Tree Diagrams

Systematic counting using the multiplication principle and tree diagrams for organizing outcomes.

### 7-3 Permutations

Arrangements of objects where order matters: $nPr = \frac{n!}{(n-r)!}$.

### 7-4 Combinations

Selections where order does not matter: $nCr = \frac{n!}{r!(n-r)!}$.

### 7-5 Compound Probability

Independent and dependent events, $P(A \text{ and } B)$, $P(A \text{ or } B)$, and the inclusion-exclusion principle.

### 7-6 Expected Value and Fair Decisions

Calculating expected value to evaluate games, insurance, raffles, and decision-making.

## Skills Developed

- Computing basic and conditional probabilities
- Applying counting principles systematically
- Distinguishing permutations from combinations
- Analyzing compound events
- Using expected value for decision-making
- Evaluating fairness in games and processes
