# Unit 8: Relations and Functions

## Overview

Students develop understanding of relations, functions, domain and range, function notation, linear vs nonlinear functions, transformations of functions, and comparing functions across multiple representations.

## Lessons

### 8-1 Relations vs Functions

Determine whether a relation is a function using tables, mappings, and the vertical line test. Identify domain and range from various representations.

### 8-2 Domain and Range

Find domain and range from graphs, tables, equations, and word problems. Recognize restrictions from square roots and denominators.

### 8-3 Function Notation f(x)

Interpret and evaluate function notation. Find inputs and outputs. Connect f(a) = b to points on a graph and real-world meaning.

### 8-4 Linear vs Nonlinear Functions

Distinguish linear from nonlinear functions using rate of change, tables, graphs, and equations. Recognize constant vs variable rates of change.

### 8-5 Function Transformations

Graph vertical shifts, horizontal shifts, reflections, and stretches of parent functions. Describe transformations from equations and graphs.

### 8-6 Comparing Functions

Compare functions represented as graphs, tables, equations, and verbal descriptions. Analyze rate of change, intercepts, and maximum/minimum values.

## Skills Developed

- Determining whether a relation is a function
- Finding domain and range from multiple representations
- Using and interpreting function notation f(x)
- Identifying linear vs nonlinear functions
- Graphing function transformations
- Comparing functions across representations
