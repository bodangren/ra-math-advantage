# Module 8 Class-Period Plan

This document is the audited class-period breakdown for Module 8.

It should be treated as the canonical planning source for Module 8 class periods unless a later audit supersedes it.

It follows the planning model in `curriculum/course-spec.md`:

- the **class period** is the atomic planning unit
- `instruction` days must map to `conductor/course-objectives.md`
- `instruction` days must name the specific worked examples they use
- `mastery`, `jigsaw`, `review`, and `test` days are marked primarily by day type

## Notes

- This is a **planning model**, not a literal calendar.
- Module 8 has 10 course objectives and 10 instruction days, so the module maps cleanly to a one-objective-per-day instruction spine.
- Lesson `8-4` is the densest lesson in the module and naturally splits into three distinct instructional days: probability distributions, normal distributions/Empirical Rule, and z-values/standard-normal analysis.
- This plan has been checked against the current repaired lesson source files and worked-example numbering.

## Why This Split Works

- `8-1` separates sampling/bias from study-type classification and inference.
- `8-2` separates direct probability comparison from simulation-based model checking.
- `8-3` functions naturally as a single descriptive-statistics day.
- `8-4` carries three distinct conceptual clusters and therefore deserves three instruction days.
- `8-5` separates confidence intervals for means from confidence intervals for proportions.
