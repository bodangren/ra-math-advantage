# Module 7 Class-Period Plan

This document is the audited class-period breakdown for Module 7.

It should be treated as the canonical planning source for Module 7 class periods unless a later audit supersedes it.

It follows the planning model in `curriculum/course-spec.md`:

- the **class period** is the atomic planning unit
- `instruction` days must map to `conductor/course-objectives.md`
- `instruction` days must name the specific worked examples they use
- `mastery`, `jigsaw`, `review`, and `test` days are marked primarily by day type

## Notes

- This is a **planning model**, not a literal calendar.
- Module 7 contains 12 course objectives and 13 instruction days, so one objective cluster intentionally spans two days.
- The extra instructional space is used in Lesson `7-6`, where routine rational-equation solving is separated from application-heavy rational-equation problems before moving to inequalities.
- Complex fractions appear in both `7-1` and `7-2`; this plan treats `7-2` as the main home for objective `7d`.
- This plan has been checked against the current repaired lesson source files and worked-example numbering.

## Why This Split Works

- `7-1` and `7-2` divide naturally into operations-focused day pairs.
- `7-3` separates reciprocal-table analysis from transformation and graph-writing work.
- `7-4` separates vertical/horizontal asymptote analysis from oblique/discontinuity behavior.
- `7-5` separates direct/joint variation from inverse/combined variation.
- `7-6` needs three instruction days: one for routine rational equations, one for applied equation problems, and one for inequalities.
