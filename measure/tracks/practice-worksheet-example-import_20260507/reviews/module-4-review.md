## Lesson Progression Assessment

The module follows a strong instructional progression:

1. **Lesson 4-1** — Graphing Linear Functions (foundational: tables, intercepts, graphing by hand)
2. **Lesson 4-2** — Rate of Change and Slope (computes slope from points/graphs, introduces slope formula)
3. **Lesson 4-3** — Slope-Intercept Form (writes and graphs [ y = mx + b ], connects slope and intercept algebraically)
4. **Lesson 4-4** — Transformations of Linear Functions (applies translations, dilations, reflections to [ f(x) = x ])
5. **Lesson 4-5** — Arithmetic Sequences (connects linear functions to discrete sequences via constant rate of change)
6. **Lesson 4-6** — Piecewise and Step Functions (extends from single linear rules to multiple rules over intervals)
7. **Lesson 4-7** — Absolute Value Functions (new function family with V-shape, applying transformations in a new context)

**Verdict:** The progression is logical and well-scaffolded. The module moves from concrete graphing skills through algebraic forms to transformations, then extends into sequences, piecewise functions, and absolute value functions. Each lesson builds prerequisite skills for the next.
